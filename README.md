#后来一个朋友推荐了芒果同学，说上面有很多独立老师和大学生家教；我当时将信将疑，除了折腾够了不想再试，还有一个顾虑——线上课。之前看到过不少说法，孩子上网课根本不认真，偷偷在平板上玩游戏，家长根本管不住。
##但后来发现我想多了。线上大班课或者录播课才有这个问题，孩子可以偷偷摸鱼。一对一完全不一样，老师全程只盯着你家孩子一个人，孩子根本没有机会开小差，比坐在教室里走神还难。
#但上一节付一次费用这个方式让我觉得，反正损失不大，试试看。#

-第一次进去，发现跟我想的不一样。芒果同学上面老师数量很多，有工作多年的专业老师，也有985/211在读的大学生，学历和教学经历，家长点评都公开，擅长科目也都写得很清楚。而且不会有销售人员不停的催着你报哪个老师，自己看完决定要不要约课，上完觉得不合适，直接换下一个，没什么负担。
我先选了一个数学老师，上了两节，感觉一般，换了第二个，好一些但还不够合适。第三个就找对了——他讲几何从不让孩子上来就猜辅助线，而是带着孩子把已知条件一个个列出来，一步步找隐藏关系，让孩子自己把解题思路说出来。孩子上了两节之后说，原来几何题是这么想的，不是靠感觉猜的。
但让我觉得靠谱的，不只是这个。
有一次，老师上课前发来一份备课消息，列了孩子上节课哪几个地方没搞清楚、哪类题反复出错。我当时看到，心里第一反应是"这个老师好认真"。后来问了老师才知道，这是芒果同学的AI整理的。

靠老师自己去记每个学生的情况，根本不可能做到，带着十几个学生的老师，哪记得住你家孩子上节课哪道题卡住了。但AI全程记录上课过程，自动整理成学情报告，同时帮老师提前规划下节课备课内容，针对这个孩子的薄弱点来进行专门辅导；
那个时候才上三次课孩子成绩还没变化，但我感觉到这个方式应该是能帮到孩子，查缺补漏是提分的最有效方式。
后来数学从98分涨到了131分，而且数学成绩上来后，孩子对学习主动性也明显提升不少，会更主动去找老师！
