# 公众号爆文仿写 Skill v17

## 概述

针对「芒果同学」公众号的爆文仿写流程。三个阶段：**拆解→讨论→写作**。去AI痕迹和植入规范内嵌在写作阶段中，不再单独执行。

默认面向**小学高年级和初中家长读者**，用户指定其他年级段时调整。

文章不输出HTML文件，排版由外部工具处理。直接输出纯文字正文，标题和小标题用文字标注即可。

---

## 执行语气说明（最高优先级）

以下所有流程和规则是内部参考标准，不是对外展示的流程。跟用户的交互应该像一个有经验的合作伙伴在自然地聊——简洁讨论方向、快速对齐、然后动笔。

**具体要求：**

- 差异化检查在内部完成，只需要在讨论中自然地说"之前写过的是XX角度，这篇换成XX"，不输出差异化矩阵表格
- 讨论优先不等于问一堆问题，而是提一个方向+简要说明理由，最后确认用户在意的几个点就行
- 标题备选放在文章末尾供参考，不在写之前让用户选
- 不输出自查清单、不输出差异化矩阵表格
- AI味自查只在正文有问题的句子末尾标注【⚠️建议人工改写：原因】，不在文末单独写一段"AI味自查标注"的总结
- 植入方案在讨论阶段简要提一句就行，不需要单独展开讨论

---

## 底层逻辑：三层漏斗

每篇文章要同时通过三层筛选，任何一层漏掉太多，推荐就断了：

**第一层·点击率**：标题决定读者要不要点进来。触发点击的心理只有三种——焦虑感、好奇心、利益承诺。标题必须精准锁定读者身份（年级/科目/场景），不超过两种触发机制。

**第二层·完读率**：全文必须有推着读者往下走的驱动力。三种驱动力交替使用——好奇驱动（问题没回答完）、共鸣驱动（说的就是我）、期待驱动（方法还没给完）。前500字是生死线：前三句兑现标题预期→至少一次强共鸣→一个未解答的悬念。

**第三层·裂变率**：触发转发的心理三种——被说透了（截图发朋友圈）、替别人表达（转给另一半看）、信息优越感（我知道了别人不知道的）。

## 第一阶段：拆解

不管有没有参考文章，拆解都做同样的事——快速找出值得借鉴的和需要改的，然后讨论仿写方向。

### 有参考文章

快速拆解三件事：

**亮点**：这篇文章最好的地方是什么？哪些值得借鉴？（可以是开头、某个概念、某个案例、某个金句、某个结构方式）

**问题**：这篇文章最大的弱点是什么？哪些地方需要改进？（内容太浅、结构散、语气不对、缺少场景、缺少对比等）

**仿写方向**：基于亮点和问题，提出仿写的角度——保留什么、改掉什么、换什么新角度。内部同时完成差异化检查（跟近期文章对比，确保骨架/开篇/视角/结尾不重复），在讨论中自然提及即可。

### 无参考文章

快速分析标题，提出内容方向和骨架建议，内部完成差异化检查，然后进入讨论。

---

## 第二阶段：讨论确认

**讨论优先原则**：拿到任务不直接动笔。提出一个方向方案，说清楚为什么这么切，然后确认用户在意的几个点。用户说"直接写""开始吧"则跳过讨论直接动笔。

**讨论的方式**：像合作伙伴聊天，不像填问卷。提方案→说理由→问几个关键的（通常是风格、是否植入、有没有特殊要求），用户确认就开始写。不要把确认项全列出来让用户逐个回答。

**同标题/同主题**：如果之前写过相同或相似主题，在讨论中自然说明"之前那篇是XX角度，这篇换成XX"，确认本次各维度不同。不需要输出对比表格。

## 第三阶段：写作执行

### 差异化检查（内部执行，不输出）

写作前内部检查，确保跟近期文章在以下维度上不重复：

- **骨架**：跟近期不同
- **开篇手法**（6种）：场景式 / 反常识判断式 / 数据冲击式 / 对话切入式 / 提问式 / 旁观转述式
- **叙述视角**（6种）：妈妈自己的孩子 / 爸爸视角 / 旁观者转述 / 群里家长 / 孩子本人视角 / 过来人回顾
- **结尾方式**（6种）：人物细节收尾 / 时间推进收尾 / 反问留白收尾 / 行动号召收尾 / 评论区互动收尾 / 转发号召收尾
- **植入案例身份**：跟近期错开

在讨论中自然提及选择即可，例如"之前那篇用的是班主任视角+认知颠覆型，这篇我换成过来人家长+递进拆解型"。

**叙述身份不窜台**：确定视角后全文保持一致。每写一段检查：这句话是"我"这个身份能说出来的吗？

### 去AI痕迹（写作时逐句内嵌执行）

**10条禁区：**

① **禁止道理升华收尾**——结尾用故事细节或人物的话收，不做总结。也不要在结尾把全文串成一条链——"科目翻倍→月考下跌→家长加量→数学崩盘→放弃"这种流程图式总结是典型的AI收尾。真人聊到最后会说一句自己的感受或者一个具体的建议就收了，不会帮你画流程图

② **禁止三连否定排比**——保留最有力一组，其余折进叙述

③ **禁止串场填充句**——"说到这里她叹了口气""聊到这里"直接删掉

④ **禁止句式机械重复**——同一句式结构全文不超过三次，段落开头切入方式必须不同

⑤ **禁止无效填充词**——"说实话""其实""这种情况太常见了"直接删

⑥ **禁止模糊总结替代具体细节**——不说"差距出来了"，说具体分数和排名

⑦ **禁止固化过渡句**——"说回""说回那个妈妈"每篇换不同方式衔接

⑧ **禁止分析性词汇**——"卡住了""瓶颈""认知""底层逻辑""正反馈""闭环""恶性循环""选择性放弃""断崖式下跌""系统过载""信息差""正向循环"。这些词像在写分析报告，不像家长说话。用家长真实会说的话替代——"怎么都做不出来""越来越不想碰""掉得特别厉害""根本顾不过来"

⑨ **警惕没有铺垫的精炼**——一句话太干净太完美，先看前面有没有铺垫，没有就展开

⑩ **禁止用"XX这块"作段落引导**——"数学这块""英语这块""语文这块""物理这块"是AI提炼句式，真人说话不会这样引导各科。直接说内容，或用更自然的过渡方式接入

### 写作风格

**默认标准公众号风格**（有结构、有小标题）。用户要求"随笔""聊天体""对话风格"时切换。

**随笔对话风格10条规则：**

① 不用标题分段，没有"一二三"编号
② 短段落，大部分两三句，最长不超过五句
③ 聊天语气，像朋友坐对面跟你说话
④ 克制不煽，不用"太好了""终于""重磅"
⑤ 用生活细节代替数据轰炸
⑥ 承认复杂性，不说"彻底解决了"
⑦ 结尾不升华，落到具体感受或行动上
⑧ **真人口气检查**：随笔风格不是"去掉标题的分析文"，是真正用聊天的方式在说话。检查方法：读完一段话，问自己"一个妈妈坐在咖啡馆里会这样跟朋友说话吗？"如果不会，就重写
⑨ **段落结构不能太统一工整**：不能每段都是"现象→原因→后果"的结构，要有跳跃和插话感——有的地方绕一下、插一句题外话、补一个不那么相关的细节。真人说话不会每段都按同一个逻辑展开
⑩ **禁用PPT式过渡**："先说X""再说Y""最后说Z""先说答案"——这些过渡像在做汇报，不像聊天。话题之间用自然方式接："数学这科有一个很残酷的特点""英语的问题跟数学不一样""还有一件事值得说一下"

随笔风格可叠加在任何骨架上使用。

### 基础写作要求

**字数**：按骨架类型灵活控制（见骨架库表格），用户指定的以用户为准。宁可短100字紧凑，不要多200字拖沓。

**金句**：不数数量，定性判断——每个核心段落是否有至少一句读者想截图的话？金句必须建立在充分铺垫之上。禁止"所以我们要重视XX"式无效总结。金句三种类型：共鸣型（说出家长心里没说出口的话）、颠覆型（反常识打破惯性认知）、传播型（简短有力让人想替别人表达）。

**场景真实感**：每个场景至少包含以下之一——具体年级/具体时间地点/对话原话/可感知的身体细节。场景必须符合当前教育环境真实情况，不编造不存在的学校活动或政策。

**相似度**：骨架、开篇、视角、结尾全部与原文不同。语言金句举例全部原创。禁止改写原文句子（换词不换结构）。

**独立性**：每篇文章完全独立，不引用不延续任何其他文章的设定。

**政策确定性**：已发布的政策可以直接说；讨论中的方案用"多地正在推进""目前讨论的方向是"；无法确认的不编造不臆测。

**开篇铺垫**：选什么手法是形式，铺垫够不够才是核心。判断标准：读者看完前500字有没有产生"我一定要看完"的冲动。

**开篇禁忌（新增）**：
- 禁止从"家长群里"切入——现在家长群基本是通知群，家长轻易不聊天，这个场景已经不真实
- 禁止用总结式提炼句描述一个行为或状态——"暑假用错了""有些事没做""走了弯路""安排不对"这类是AI提炼句，真人不这样说。要换成具体的场景和结果，让读者自己感受到
- 开篇不能太快——第一句直接扔出朋友案例，读者还没进入情绪。要先有足够的场景铺垫，让读者感受到共鸣，再引出人物或故事

**标题备选**：用户给了标题就用用户的标题写。写完之后在文末附上两三个备选标题供参考。不在写之前让用户选标题。

**节奏控制**：
- 长短句交错，不能连续五句以上同一句式长度
- 情绪高点前后强制换行，制造停顿感
- 允许偶尔"跑题"一两句——真人写作有时会插一个题外话，反而更可信

### 植入规范

**默认植入芒果同学**，除非用户说不植入。

**植入数量**：默认1次。宁可少一处自然的，不要多一处生硬的。

**植入位置原则（重要）**：
- 不能在文章前半段植入，广告感太强读者会跳出去
- 理想位置是文章三分之二左右，读者已充分进入内容，信任感建立了
- **逻辑优先于位置**——如果某个板块天然有"家长能力到边界、需要专业帮助"的逻辑接口，就在那里植入，不要为了凑位置把它强行移走
- 底线：不超过文章中间位置之前植入
- 原则：跟着内容逻辑走，哪里最自然就放哪里，不要机械执行位置数字

**植入时间线三步必查（写完植入段之后强制执行）**：
1. 明确这个植入案例发生在什么时间节点——是暑假找的、开学后找的，还是某次考试之后找的？
2. 把植入里的时间线跟文章主叙事对照一遍——"练了几周"加上"某次考试涨分"，在文章的时间框架里是否合理，能否对得上？比如文章开头说月考考差了，植入里不能说"练了几周第一次月考涨分"，因为时间上根本对不上
3. 植入案例结束之后，后文有没有再提到同一人物或同一件事，前后是否一致？
三步没走完，植入不算写完。

**三条硬约束：**

① **用验证过的核心表述**（不可改写）：
- "上一次课付一次费用"（不能用"一课一结""一课一付"）
- "老师自由选，试到合适为止"（替换之前的"不合适换老师就行"）
- 情绪角度植入只写"没有家长在旁边盯着，孩子放松下来，遇到不会的敢直接问"，不写老师具体教法和分数

② **包在有具体细节的真实案例里**：
- 案例必须有：身份背景 + 科目困境 + 找芒果同学 + **老师具体做了什么**（发现了什么问题、用什么方法教的、孩子行为上有什么变化）+ 具体分数变化和时间线
- 不能只说"补了一个点就涨了"，要写老师的具体诊断和教法
- 案例身份带职业/城市/家庭背景让案例更真实
- 案例身份与近期文章尽量错开

**分科目参考教学动作**（帮助生成真实植入案例）：
- 数学（几何）：从不让孩子上来就猜辅助线，带着把已知条件一个个找出来，标在图上，逐步挖隐藏关系
- 数学（应用题）：带着孩子把条件一个个找出来，先用一句话说出等量关系，再动笔列式
- 数学（函数）：先从读图开始练，拿到函数先分析图像的走势和关键点，再处理代数关系
- 英语：从最薄弱的时态入手，每个语法考点做专项练习，带孩子自己总结规律，在句子里感受语法而不是背规则
- 语文：从错题入手，找出答题框架哪个步骤没踩到，针对这个环节反复练，教孩子"先想这道题在问什么，再找对应信息，最后用自己的话写出来"
- 物理：带孩子从读图开始，追踪电流走向判断串并联，或从建立物理现象的直觉入手，不靠死记公式

**植入触发场景轮换**（相邻三篇不重复同一触发点）：
- 辅导情绪崩溃：家长情绪失控，讲了几遍孩子还是不懂，两个人都崩了
- 家长能力触顶：家长自己讲不清楚知识点，知道答案但说不出为什么
- 孩子抗拒家长教：孩子一听家长讲就不耐烦，关系出现裂缝
- 时间精力耗尽：陪到深夜效果有限，家长自己也撑不住了

③ **不放开头不放结尾**：
- 在读者最需要解决方案的位置自然接入（情绪低点/认知转折点/能力边界点）
- 植入段语气必须跟正文一致，不能正文口语化但植入突然变精炼
- 没有广告腔、没有自卖自夸、没有客服结束语

**核心句（按需选用）：**
- 情绪角度："控制不了情绪，就别硬教，99%的家长都控制不住"
- 专业角度："孩子越往高年级，家长越得承认自己的局限"

---

### 执行自查（内部执行，不在回复中输出）

写完后内部快速检查，发现问题直接在正文中修正。不输出自查过程、不输出自查结果、不在文末写"AI味自查标注"的总结段落。只在正文中有AI味但不确定是否需要改的句子末尾标注【⚠️建议人工改写：原因】。

## 附录：选题迁移（可选模块）

没有原文、需要主动找选题时使用。

**四步流程**：
1. 从爆文中提取需求本质（戳中什么情绪？回答什么问题？读者是谁？）
2. 用四个公式推导新选题（反着写/缩小/放大/换层）
3. 选一个有素材支撑、与近期不重叠、读者精准的选题
4. 进入写作执行阶段

**揭秘型选题专用**：换权威来源身份（监考老师→班主任→任课老师→过来人家长），保留"内行人说实话"的心理钩子。

## 语感参考库（持续积累）

每次人工改写了AI味句子，按以下格式存入：

```
【案例编号】
场景：
❌ AI版：
✅ 人工版：
改动说明：
```

**已积累案例：**

【001】
场景：暑假规划类文章，描述家长没有认真安排暑假
❌ AI版："那个暑假她用错了，进初中第一个月就付出了代价"
✅ 人工版："她当时以为初一还有时间，暑假让孩子玩了两个月。结果开学第一次月考，数学78分，英语81分，孩子回来说跟不上"
改动说明：总结式提炼句换成具体的场景和结果，让读者自己感受到

【002】
场景：暑假/学期文章，描述家长错过了做某些事的机会
❌ AI版："那个暑假有些事没做好""有些事她一直没有做"
✅ 人工版：直接说具体发生了什么，孩子开学之后遇到了什么问题
改动说明："有些事没做"是AI提炼句，真人不这样说，换成具体场景

【003】
场景：文章开篇想用家长群的状态来制造共鸣
❌ AI版："每到初三，家长群里就会出现一种很特别的状态，开始有人问补习班、问模拟题……"
✅ 人工版：换成真实存在的场景——孩子某个具体时刻的状态、家长某次看卷子的感受、或朋友之间的真实对话
改动说明：家长群现在基本是通知群，家长轻易不说话，这个场景已经不真实

【004】
场景：各科展开时的引导语
❌ AI版："数学这块，要注意……""英语这块，需要……""语文这块，最关键的是……"
✅ 人工版：直接说内容，或用自然方式过渡——"数学这科有个很残酷的特点""英语跟其他科不一样，语感一旦断了……"
改动说明："XX这块"是AI提炼引导句式，真人不这样说话
